## 🎯 Common Tasks

### Change User Data
- Edit: `src/data/dummyData.js`
- Search: `dummyUserData`
- Modify: name, blood group, allergies, etc.

### Change Colors
- Edit: `src/index.css`
- Find: `background: linear-gradient`
- Change: hex color codes

### Add New Page
1. Create: `src/pages/MyPage.jsx`
2. Add route in: `src/App.jsx`
3. Add nav item in: `src/components/Navbar.jsx`

### Connect to API
1. Create: `src/services/api.js`
2. Replace: dummyData with API calls
3. Add environment variables

### Deploy
1. Run: `npm run build`
2. Upload: `dist/` folder
3. Or: Push to GitHub → Connect to Vercel
